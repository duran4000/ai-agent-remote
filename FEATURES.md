# AI Agent Remote - 功能列表

## 已完成功能

### 核心功能
- [x] WebSocket 远程终端连接
- [x] 多标签页支持
- [x] 终端自适应大小
- [x] 会话状态管理

### 终端功能
- [x] 终端滚动位置保持（AI输出时不再跳回底部）
- [x] 终端日志导出
- [x] 终端内搜索（🔍按钮）
- [x] 命令历史导航（上下箭头）

### 连接与状态
- [x] 连接质量指示（指示灯 + 毫秒数，颜色联动）
  - 绿色: 极好 (<100ms)
  - 黄绿色: 良好 (100-200ms)
  - 黄色: 一般 (200-500ms)
  - 红色: 较差 (>500ms)

### 通知
- [x] AI 完成通知（浏览器通知）
  - 状态: 代码已完成，但通知权限可能被浏览器拒绝
  - 检测时间: 0.5秒无输出认为完成

### 国际化
- [x] i18n 国际化支持（中英文切换）
- [x] 管理页面国际化
